## 📋 Descripción del Proyecto
Este repositorio contiene la resolución del **Examen 3** correspondiente al curso de **Desarrollo de Aplicaciones en Internet**. El proyecto se enfoca en la implementación de lógicas avanzadas de backend, validación de datos y manejo de arquitecturas web siguiendo las mejores prácticas de la industria.

## 🛠 Tecnologías Utilizadas
| Herramienta | Descripción |
| :--- | :--- |
| **PHP** | Lenguaje principal para la lógica del servidor. |
| **HTML5 / CSS3** | Estructuración y diseño de interfaces responsivas. |
| **MySQL** | Gestión de bases de datos relacionales. |
| **Git** | Control de versiones para el seguimiento del examen. |

## 🚀 Características Principales
- [x] **Validación Dinámica:** Implementación de reglas de negocio en el lado del servidor.
- [x] **Seguridad:** Filtrado de datos para evitar vulnerabilidades comunes (XSS/SQLi).
- [x] **Arquitectura:** Estructura modular para facilitar la escalabilidad del código.
- [x] **Interfaz:** Diseño centrado en la experiencia de usuario (UX).
